# Games in Raylib - C/C++

Games created using Raylib library.

## Break-It

This game is one of the GD50 project. It is inspired from classic game **Break Out**.

### Features

- Written in **C**
- Spritesheet texture
- Collision detection
- Emitters/Particles System
- Different game states
- Procedural generation of each level
- Keyboard control
- Sound effects
